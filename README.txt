Run the game first

Copy the "g1g2_jump.mp3" file into the following folder:

  > .../Implementation/AwesomePokerGameSln/bin/Debug
  
Next time you run, when you press the start game button music will begin to play


** If given an error about WMPLib:

  1. Double click this file: FrmPlayGame.cs
  
  2. Right click on any item within the toolbox to the left

  3. Select "Choose Items..."
  
  4. Go to the "COM Components" tab
  
  5. Make sure "Windows Media Player" is checked
  
  6. Drag Windows Media Player onto the FrmPlayGame.cs [Design] view
  
  7. Delete the Windows Media Player, re-run with no issues
